# Документация проекта Docs Q&A RAG

Добро пожаловать в документацию приложения Docs Q&A RAG.

## Обзор
Приложение Docs Q&A RAG ("Docs Question & Answering Retrieval-Augmented Generation") позволяет пользователям загружать текстовые документы (PDF, TXT) и задавать вопросы по их содержимому, получая точные ответы с указанием источников. Система использует передовые технологии искусственного интеллекта для анализа и поиска информации.

## Ключевые возможности

### 1. Управление документами
*   **Загрузка**: Поддержка форматов `.pdf` и `.txt`.
*   **Индексация**: Автоматическое извлечение текста, разделение на фрагменты (чанки) и создание векторных представлений (эмбеддингов).
*   **Список документов**: Просмотр статуса обработки (`uploaded`, `indexing`, `indexed`, `failed`).
*   **Удаление**: Возможность удалить документ и все связанные с ним данные из базы.

### 2. Умный поиск и ответы (RAG Pipeline)
*   **Семантический поиск**: Система ищет ответы не по ключевым словам, а по смыслу, используя векторную базу данных.
*   **Генерация ответов**: Найденный контекст передается в LLM (Large Language Model), которая формирует связный ответ на естественном языке.
*   **Прозрачность**: Каждый ответ сопровождается списком источников (имя файла и конкретный фрагмент текста), на основе которых он был сформирован.

### 3. Пользовательский интерфейс
*   **Streamlit**: Интуитивный веб-интерфейс, не требующий специальных знаний для использования.
*   **Чат**: Привычный формат диалога для взаимодействия с системой.

## Структура проекта
Описание файлов и папок: [Project Structure](project_structure.md).


*   **Backend**: [FastAPI](https://fastapi.tiangolo.com/) — высокопроизводительный асинхронный веб-фреймворк.
*   **Frontend**: [Streamlit](https://streamlit.io/) — библиотека для быстрого создания data science приложений.
*   **База данных**: [Supabase](https://supabase.com/) (PostgreSQL) с расширением `pgvector` для хранения векторов.
*   **AI Модели**:
    *   **Embeddings**: OpenAI `text-embedding-3-small` (для превращения текста в векторы).
    *   **LLM**: OpenAI `gpt-5` (для генерации ответов).
*   **Язык**: Python 3.9+
